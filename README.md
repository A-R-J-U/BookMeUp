## üéâ To Run

1.**Clone the repository:**

   ```bash
   git clone https://github.com/A-R-J-U/BookMeUp
   ```

2.**Specify .env files**

*Frontend*:

```bash
    PORT = ...

    BASE_URL = ...
    FRONTEND_URL = ...

    #Database Configuration
        DB_NAME = ...
        DB_HOST = ...
        DB_PORT = ...
        DB_USER = ...
        DB_PASS = ...

        JWT_SECRET=...


    #SMTP Configuration
        EMAIL_NAME=...
        EMAIL_PASSWORD =...
```
*Backend*:
```bash
   VITE_API_URL=
```
3.**install dependencies**

*in root folder of frontend and Backend*
```bash
   npm install
```
4.**Start**

*Run both the frontend and Backend using the following command*
```bash
   npm run dev 
```
